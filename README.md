# kafka
Приложение producer отправляет сгенерированные сообщения в топик sensor_data в брокер сообщений из 3 нод<br>
Два консьюмера читают consumer и consumer2<br>
Первый консьюмер (consumer) считывает по одному сообщению, обрабатывать его и коммитить оффсет автоматически<br>
Второй консьюмер (consumer2) считывает минимум по 10 сообщений за один poll, обрабатывать сообщения в цикле и один раз <br>
